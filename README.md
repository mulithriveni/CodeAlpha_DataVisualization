# CodeAlpha - Data Visualization

## Objective

The objective of this project is to transform raw sales data into meaningful visual insights using Power BI. Data visualization helps in understanding trends, patterns, and business performance more effectively.

## Tools Used

* Power BI Desktop
* CSV Dataset
* Data Visualization Techniques

## Dataset

Sales Data Dataset containing:

* Month
* Region
* Category
* Sales
* Profit

## Visualizations Created

1. Sales Trend by Month (Line Chart)
2. Sales by Region (Bar Chart)
3. Profit by Category (Pie Chart)
4. Total Sales KPI Card
5. Total Profit KPI Card

## Dashboard Features

* Interactive and user-friendly dashboard
* Region-wise sales analysis
* Category-wise profit analysis
* Monthly sales trend visualization
* Key Performance Indicators (KPIs)

## Key Insights

* Electronics category generated strong sales performance.
* Sales varied across different regions.
* Monthly sales showed a consistent trend throughout the year.
* Profitability differed among product categories.
* Dashboard visualizations simplified business decision-making.
